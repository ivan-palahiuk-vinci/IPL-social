Explication du code:
Tout d'abord, je mets en place une pipeline qui va tester mon code à chaque pull request. Ensuite, je crée un fichier de test pour ma fonction de validation d'e-mail. Une fois les tests mis en place, j'implémente les vérifications nécessaires dans ma méthode afin de réussir les tests 1 à 1 ; j'utilise la méthode TDD. Pour finir, une fois tous mes tests réussis, je peux lancer une pull request vers ma branche main. La pipeline va lancer les tests lors de la pull request et vérifier que tous les tests réussissent.

Commande pour lancer les tests: npm run test
